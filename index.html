Configurar / reconfigurar WiFi do controlador
Primeiro uso: ao ligar o controlador, ele cria a rede WiFi Irrigacao-esp32_lavoura01.

1. Conecte o celular nessa rede (senha: irrigacao123)
2. Abra o navegador em 192.168.4.1
3. Preencha o nome e senha do WiFi da lavoura
4. O controlador reinicia conectado

Para trocar o WiFi depois: use o botão abaixo.
Forçar reconfiguração via MQTT
Envia comando para o controlador entrar em modo portal WiFi.
✅ Conexão pública
O broker está acessível publicamente via Tailscale Funnel com certificado válido (Let's Encrypt). Sem necessidade de VPN ou instalação de certificado em cada dispositivo.

Basta abrir esta página em qualquer navegador (PC ou celular) e tocar em Conectar.
